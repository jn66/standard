## 简介
这是一份中国知网前端参考文档。内容涵盖项目的开发流程和前端开发的基本规范。意在建立项目开发的基本蓝图。解决开发中常见的问题，提高项目的可维护性，建立统一的标准规范。

## 使用范围
本参考文档广泛使用于中国知网“赛事星PC”、“赛事星WAP”、“手机知网”三大项目中。
